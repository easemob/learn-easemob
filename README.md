# 开源项目简介：

环环教育demo，覆盖1对1互动教学、在线互动小班课两种核心教学场景，具备即时消息互动、实时音视频互动、互动白板、屏幕共享等丰富功能。Demo生动展示了如何用IM、音视频、互动白板SDK共同搭建在线互动教学场景。同时demo覆盖了教师端和学生端，并开发实现了iOS、Android、Web三端应用。现在正式开源给小伙伴们，详细介绍请往下看。

## 核心功能
- 1对1互动教学
- 在线互动小班课（1名老师+多位学生互动）
- 即时消息互动（聊天室）
- 实时音视频互动（音视频）
- 互动白板
- 白板权限控制
- 屏幕共享
- 学生列表（小班课老师端特有）
